Dosage& Formulation

Different types of dosage forms
Medicine Type APIs Excipients Dosage Form Formulation Technique
Tablets Ibuprofen, Paracetamol and Aspirin Binders(cellulose), Fillers(lactose), Disintegrants(crospovidone), Lubricants(magnesium stearate), Preservatives(paraben), Flavoring Agents(menthol), Colorants(FD C dyes) Dosage form: Solid dose form containing API and excipients,often coated for ease of swallowing Granulation and Compression
Capsules Amoxicillin, Metformin, Simvastatin Gelatin(shell), Fillers(microcrystalline cellulose), Disintegrants(sodium starch glycolate), Lubricants(magnesium stearate), Preservatives(sodium benzoate), Flavoring agents(artificial flavors), Colorants(FD C dyes) Gelatin or vegetarian capsules containing the API in powder in powder or liquid form Encapsulation
Solutions Paracetamol, Amoxicillin, Ciprofloxacin Solvents(water and alcohol), Preservatives(sodium benzoate), Flavoring Agents(menthol), Colorants(FD C dyes) Liquid dosage forms where the API is dissolved in a solvent Dissolution
Suspension Amoxicillin, Ibuprofen, Ciprofloxacin Suspended particles(finely divided API), Dispersing agents(sorbitol), Preservatives(sodium benzoate), Flavoring agents(menthol), Colorants(FD C dyes) Liquid dosage forms where the API is suspended in a liquid, requiring shakage before use Suspension
